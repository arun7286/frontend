Variables which are declared using LET will be stord in side Temporal Dead Zone.

The TDZ is the area of memory where uninitialized variables will be stored.

If programmer will access the variable which are inside TDZ. will get reference Err.

The variable which are declared using LET cannot be accessed before its declaration.
